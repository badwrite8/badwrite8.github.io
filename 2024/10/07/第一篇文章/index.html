创建 hexo

1.全局安装

npm install -g hexo-cli

2.初始化 hexo

新建文件夹下hexo init

（其实相当于使用 git clone 对应 github 地址）

3.打开 hexo 服务本地运行

hexo g

hexo s

4.创建文件

hexo new "文章标题"

在 source–>__posts 可以看到

5.更新

hexo g

hexo s

6.推送

hexo clean

hexo g -d 部署
